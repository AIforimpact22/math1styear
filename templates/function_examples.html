These curated mappings demonstrate what it means to be a function in the geology-themed coursework. Each source value is paired with exactly one output, making the vertical-line test succeed.

Pairs are convenient reminders that functions can output vectors, not just scalars.

Checklist for a function

  • Every input appears at most once in the left column.
  • No input is paired with two different outputs.
  • Outputs may repeat — surjective/injective tests come later.
  • Visual: every vertical line intersects the graph at most once.

Classroom use

Start with the geothermal mapping, then contrast with a non-function (e.g., two lithologies for one well) before returning here to reinforce the rule.

Encourage students to build their own tables using local data, keeping the one-output rule in mind.

Key idea Function = deterministic mapping
Functions let us predict or model outcomes consistently. Whether we map depth to temperature, porosity to workflows, or samples to mineral fractions, the determinism is what powers analysis, automation, and grading rubrics in this course.
